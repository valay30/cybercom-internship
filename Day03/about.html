About Us

web project designed to showcase products and demonstrate core web development concepts using HTML, CSS, and basic JavaScript. This website presents various products with details such as images, features, pricing, quantity, and availability in a structured and user-friendly format.

The project aims to build a strong foundation in front-end development by implementing responsive layouts, reusable components, interactive elements, and clean UI design. Features like editable quantity, availability status, and dynamic behavior help simulate real-world e-commerce and admin panel functionality.

This project reflects a hands-on learning approach and highlights skills in layout design, styling, accessibility basics, and simple client-side logic. It is created for educational purposes and continuous skill improvement in modern web development.

Technologies Used

  • HTML5
  • CSS3
  • Git & GitHub
